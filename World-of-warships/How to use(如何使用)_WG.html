1,Open your Steam/Whagging Game Center,Select in sequence as indicated by the arrows in the diagram

一,打开你的steam/Wargaming Game center，按图中箭头所示顺序依次选择


2,Open the 'bin' folder, find the folder with the highest number and open it

二,打开'bin'文件夹,找到数字最大的文件夹并打开它


3,If your 'bin' folder does not contain the 'res_mods' folder, please create a new 'res_mods' folder If there is, open it

三,如果你的'bin'文件夹里没有'res_mods'文件夹,请新建'res_mods'.如果有，打开它


4,Put the downloaded files starting with 'banks' into the newly created' res_mods' folder and change the file name to 'banks'

四,将下载的以'banks'开头的文件放入新建的'res_mods'文件夹里，并将文件名修改为'banks'


5,Open the game, select the menu in the upper left corner, and choose Settings

五,打开游戏，选择左上角菜单，选择设置


6，Select the 'Sound' option on the left, slide to the bottom, and choose 'Type' to see the installed voice pack

六,选择左侧的'声音'选项，滑到最下方，选择'类型'，即可看到安装的语音包
